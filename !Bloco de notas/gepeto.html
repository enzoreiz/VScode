📚 Minhas Anotações

📌 HTML

HTML é a linguagem usada para estruturar páginas na web. Ele organiza o conteúdo através de elementos chamados “tags”, que definem coisas como títulos, parágrafos, links e imagens. Pode ser visto como o esqueleto do site, já que é a base onde tudo é construído.

🎨 CSS

CSS é a linguagem responsável pela aparência das páginas. Ele controla cores, tamanhos, fontes, espaçamentos e até animações. Enquanto o HTML monta a estrutura, o CSS dá estilo e deixa o site visualmente agradável.

⚡ JavaScript

JavaScript é a linguagem que dá vida às páginas web. Ele permite criar interatividade, validações, efeitos dinâmicos e até aplicações completas dentro do navegador. Enquanto HTML e CSS constroem e decoram, o JavaScript faz acontecer.

🌱 Git

Git é um sistema de controle de versão que permite acompanhar mudanças no código e colaborar com outras pessoas em projetos. Ele funciona como uma “linha do tempo” onde você pode salvar, voltar atrás e organizar cada etapa do desenvolvimento.
