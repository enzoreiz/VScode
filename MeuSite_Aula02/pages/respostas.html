Perguntas e Respostas

Qual é a diferença entre link interno e externo?
R: Link interno leva o usuário para outra página dentro do mesmo site. Link externo leva para um site diferente.
Qual é a diferença entre a, a:active, a:visited, e a:hover?
  • a: estilo padrão do link.
  • a:hover: quando o mouse passa por cima.
  • a:active: quando o link é clicado.
  • a:visited: quando o link já foi visitado.
Qual é a diferença entre criar listas com Ul e ol ?
  • UL: lista não ordenada (com bolinhas ou símbolos).
  • OL: lista ordenada (com números ou letras).
